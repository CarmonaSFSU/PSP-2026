PINC 2026 - Environmental Health Team

This repository contains materials used by the Environmental Health Team project. 

The analysis will include: 
Building permits: description, date, status, cost, address/parcel
Heat exposure: temperature or heat vulnerability indicators
Historical disinvestment: HOLC redlining grades
Social vulnerability: CDC/ATSDR SVI or census indicators
Housing context: building age, renter share, multifamily housing
