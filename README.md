## PINC 2026 - Environmental Health Team

Welcome! This GitHub repository will be used to share project files, including R Markdown files, examples, and data files.

## Getting started

To download the repository to your computer:

1. Open RStudio.
2. Go to **File → New Project → Version Control → Git**.
3. Paste this repository URL:

   https://github.com/CarmonaSFSU/PSP-2026.git

4. Choose where you want the project folder to live on your computer.
5. Click **Create Project**.

## Getting updates

I may update files throughout the course/project. To get the newest version, open this project in RStudio and go to the **Terminal** tab.

Run: 
>git pull

*Important workflow guidelines*

Please do not edit the original activity, example, or data files directly unless instructed.
